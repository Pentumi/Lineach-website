# Commercial Cleaning Services Landing Page

A standalone landing page for Lineach's commercial cleaning services, accessible at `https://lineach.com/cleaningservices`.

## Overview

This is a complete, self-contained landing page designed for lead generation in the commercial cleaning services market. The page is optimized for conversion with a focus on B2B lead capture.

## Features

### 🎯 Lead Generation
- **Above-the-fold lead capture form** with comprehensive fields
- **GDPR-compliant consent** checkbox
- **Multiple CTAs** throughout the page
- **Thank you page** with next steps

### 📱 Responsive Design
- **Mobile-first** approach
- **Fully responsive** across all devices
- **Optimized for tablets** and desktop

### 🎨 Modern UI/UX
- **Clean, professional design** using TailwindCSS
- **Smooth animations** and hover effects
- **Accessibility compliant** with proper focus states
- **High contrast mode** support

### 🔍 SEO Optimized
- **Structured data** (JSON-LD) for LocalBusiness
- **Meta tags** optimized for "Commercial Cleaning Services"
- **Semantic HTML** structure
- **Open Graph** and Twitter Card support

### 📊 Analytics Ready
- **Event tracking** placeholders for form interactions
- **Performance monitoring** setup
- **Conversion tracking** ready for implementation

## Deployment

### Netlify Deployment
1. Upload the entire `cleaningservices/` folder to your Netlify site
2. The page will be accessible at `https://lineach.com/cleaningservices`
3. No changes to existing site structure required

### Local Development
1. Navigate to the `cleaningservices/` directory
2. Open `index.html` in a web browser
3. Or use a local server: `python -m http.server 8000`
